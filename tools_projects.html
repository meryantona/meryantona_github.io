Cron Security Scan

This Bash script performs a comprehensive security scan of cron jobs on a Linux system. The script provides a step-by-step, interactive review of each area, using color-coded output for better readability. It's designed for system administrators and security professionals to quickly assess the cron job configuration on a system.

Processes Security Scan

This Bash script is a practical security tool designed to provide a comprehensive overview of system processes. It utilizes color-coded output and interactive pauses to guide users through various levels of process information. The script showcases effective use of system commands and Bash scripting techniques to create a user-friendly, informative security scanning tool.

Security Cleaner

Security Cleaner is a comprehensive script that helps you securely clean up various types of data from your Linux system.
